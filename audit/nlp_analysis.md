## GLOBAL NLP ISSUES

### Passive Voice Instances (must convert to active)
1. "yang seharusnya digunakan" → "yang bisa Anda gunakan"
2. "dirangkum menjadi insight" → "GeoVera merangkum menjadi insight"
3. "dikirim langsung ke WhatsApp" → "GeoVera kirim langsung ke WhatsApp"
4. "dioperasikan langsung melalui WhatsApp" → "Anda operasikan langsung via WhatsApp"

### Weak Verbs (replace with power verbs)
- "mengelola" → "kuasai" / "kendalikan"
- "membantu" → "percepat" / "dorong"
- "bekerja" → "eksekusi" / "jalankan"
- "menganalisis" → "bedah" / "identifikasi"
- "meningkatkan" → "dongkrak" / "akselerasi"

- ❌ No client/user count social proof
